# To-Do List Application

A simple command-line task manager built with Python.

## Features

- Add tasks
- View all tasks
- Delete tasks
- Input validation and error handling

## Requirements

- Python 3.6+

## Usage

Run the application:

```bash
python To-do.py
```

## Menu Options

1. **Add a task** - Create new tasks
2. **View all tasks** - Display all tasks with numbers
3. **Delete a task** - Remove tasks by number
4. **Quit** - Exit the application

## Example

```
Enter your choice (1-4): 1
Enter the task description: Buy groceries
[+] Success! Task 'Buy groceries' has been added.

Enter your choice (1-4): 2
--- YOUR TASKS ---
You have 1 task(s):
1. Buy groceries
```
